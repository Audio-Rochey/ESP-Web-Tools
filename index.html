TerrainTronics Web Installer for Chrome/Edge

  1. Plug in your ESP to a USB port. We will install Pre-Built Code to your Wemos D1 Mini 0.12.0 to it.
  2. Hit "Install" and select the correct COM port. No device found?
  3. Get code installed and connected in less than 3 minutes!
  4. For Wifi enabled code, click on "logs" - the device will show which temporary wifi to connect to, to configure it, and the Chrome IP address to go.
